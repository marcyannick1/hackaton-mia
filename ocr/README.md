# OCR

## Setup Instructions

1. **Launch your docker engine**
2. **Go to project root directory**
```shell
cd hackaton-mia
```
3. **Launch container**
```shell
docker compose up --build
```
4. **Access to OCR**  
*Access OCR with __127.0.0.1:8000__*
  

5. **View OCR API Docs and test Api**  
*Access __127.0.0.1:8000/docs__*
